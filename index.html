Timestamp microservice

Add a Unix timestamp or natural language date as a parameter to this url and this service will respond with a JSON object that contains the Unix timestamp and natural language form of the date give.

Example

https://timestamp-ms.herokuapp.com/December%2015,%202015

or

https://timestamp-ms.herokuapp.com/1450137600

Example output

{ "unix": 1450137600, "natural": "December 15, 2015" }
